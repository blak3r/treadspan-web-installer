1. Connect the ESP32 module you purchased to your computer using a USB-C Cable that supports data.
  2. Press the blue "Connect" Button below and select the port (if you don't see usb-serial, install CP2102 drivers ) - also you might try flipping the USB cable around make sure it's getting power.
  3. Once connected, choose the "Install Treadspan Web Installer". Allow it to install for about 2 minutes.
  4. After it's finished, a dialog window will come up which will give you option to Set Wifi. Do that!
